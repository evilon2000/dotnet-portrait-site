autchon_backend

订单履约后端，处理支付后库存占用、仓库分配、发货检查和取消回滚

.NET Framework State Machine Azure Function Queue Workflow

架构拓扑

项目介绍

业务场景

在订单履约场景下保持状态一致性，支持支付后流程推进、库存动作与发运联动

项目细节（节选）

  1. 订单、支付、库存三域按边界解耦
  2. 状态流转前置规则校验防止越权跳转
  3. 事件总线驱动异步通知与后置动作
  4. 关键接口提供重试与幂等保护
  5. 支持订单履约过程的对账与追踪查询

技术亮点

这个项目重点处理订单从支付、分仓、发货到完成的状态流转，并用队列执行异步任务。
